\chapter{在学期间取得的主要成果}

\subsection*{一、已发表或已录用的论文}
% 【本人为第一作者或指导教师为第一作者，学生为第二作者，分条目列举，格式如下：】

\begin{enumerate}
	\item 期刊论文格式：作者. 论文题名[J/OL]. 刊名, 出版年份, 卷号(期号): 起止页码[引用日期]. 获取和访问路径. DOI.
	\item 会议论文格式：作者. 论文题名[C/OL]//论文集主要责任者. 论文集名, 出版地（或会议地址）: 出版社，出版年份：页码. [引用日期], 获取和访问路径. DOI.
\end{enumerate}

\subsection*{二、已获专利或软件著作权}
% 【专利或软件著作权，格式如下：】

\begin{enumerate}
	\item 专利格式：专利申请者或所有者. 专利名: 专利号或公开号[P/OL], 公告日期或公开日期[引用日期]. 获取和访问路径.（本人排名）
	\item 软著格式：著作权人. 软件名称: 登记号[C/P], 登记日期. （本人排名）
\end{enumerate}
